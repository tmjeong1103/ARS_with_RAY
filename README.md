# RL_with_RAY

> Implementation of reinforcement learning algorithms that is applied RAY in PyTorch.
> 
> In this repo, we implemented this RL_with_RAY code only use RAY CORE from scratch, didn't use RLLib.
> 

## Dependencies
1. Python 3.6.x
2. PyTorch 1.8.1
3. RAY 1.3.0
4. pybullet-gym
5. numpy
